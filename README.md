# PotionScraper

## LINUX OR UBUNTU
```
git clone https://github.com/Jinx-CrazyCode/PotionScraper
pip install requests
pip install colorama
pip install beautifulsoup4
pip install pystyle
cd PotionScraper
python main.py
```
## WINDOWS

> Install and check out the repository and install the libraries
```
pip install requests
pip install colorama
pip install beautifulsoup4
pip install pystyle
cd PotionScraper
python main.py
```
